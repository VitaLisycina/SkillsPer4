# SkillsPer4

You got 7 colours cubs.

At the beginning of the game, there's a screen with a start button.
When you start, three colored cubes in a specific order appear on the screen for a couple of seconds.
When the hint gone, you press on cubes with same order of colour.
You have to repeat the color sequence exactly. If you did everything correctly, the cubes line up to form a tower. 
If not, the tower collapses and the result is displayed on the screen.
If you do it correctly twice in a row, a new cube is added to the hint.
